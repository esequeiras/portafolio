Proyectos

Bloom es un proyecto web desarrollado por un equipo de seis personas como parte del curso de Proyecto 4. Esta plataforma utiliza tecnologías como Angular y Spring Boot para proporcionar una experiencia de usuario dinámica y eficiente. Bloom cuenta con diversas secciones diseñadas específicamente para ayudar a personas que sufren de ansiedad. Cuenta con un dashboard donde las personas podrán ver sus datos, tales como nivel y ataques de ansiedad que han experimentado, una sección de juegos, foro para compartir experiencias, un diario emocional que se integró con chat GPT para proporcionar inspiracion, un calendario para que los usuarios puedan organizar su tiempo....
